Add option to specify the tag name and properties of the container element of the generated Graphviz diagram

With the introduction of the properties option, deprecate the className and style options. To specify the class name and style of the container element, use the `properties.className` and `properties.style` properties instead:

```ts
import { unified } from "unified";
import remarkParse from "remark-parse";
import remarkRehype from "remark-rehype";
import rehypeGraphviz from "rehype-graphviz";
import rehypeStringify from "rehype-stringify";
import { Graphviz } from "@hpcc-js/wasm";

const md = `
# Hello World

\`\`\`dot
digraph {
  a -> b
}
\`\`\`
`;

const html = unified()
  .use(remarkParse)
  .use(remarkRehype)
  .use(rehypeGraphviz, {
    graphviz: await Graphviz.load(),
    properties: {
      className: "graphviz",
      style: "overflow: clip;",
    },
  })
  .use(rehypeStringify)
  .processSync(md)
  .toString();

console.log(html);
```
